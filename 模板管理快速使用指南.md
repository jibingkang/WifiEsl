# 模板管理快速使用指南

## 🚀 快速开始

### 1. 访问模板管理页面
- 在侧边栏点击 **"模板管理"** 菜单
- 或访问网址: `http://localhost:5173/template/manage`

### 2. 使用现有模板文件
1. 在模板管理页面点击 **"导入JSON文件"**
2. 选择项目根目录的 `template.json` 文件
3. 系统会自动创建名为 "Test1" 的模板

### 3. 手动创建新模板
1. 点击 **"新增模板"**
2. 填写模板信息：
   - 模板ID：任意唯一标识符
   - 模板名称：显示名称
   - 描述：可选
   - 屏幕类型：单色屏/彩色屏等
3. 添加字段：
   - 点击"添加字段"按钮
   - 为每个字段设置：key、label、类型、是否必填、默认值
4. 点击"创建"

## 📁 JSON文件格式示例

### 简单格式（从项目示例）
```json
{
  "tid": "69d7c56edc4d8169e2140f99",
  "tname": "Test1",
  "data": {
    "text-0": "TEXT"
  }
}
```

### 完整格式
```json
{
  "tid": "my-template",
  "tname": "我的模板",
  "description": "这是一个测试模板",
  "screen_type": "monochrome",
  "fields": [
    {
      "key": "title",
      "label": "标题",
      "type": "text",
      "required": true,
      "default_value": "商品标题"
    },
    {
      "key": "price", 
      "label": "价格",
      "type": "text",
      "required": true,
      "default_value": "¥0.00"
    }
  ]
}
```

## 🔧 主要功能

| 功能 | 操作方式 | 说明 |
|------|----------|------|
| **查看模板** | 自动加载 | 进入页面即显示所有模板 |
| **新增模板** | 点击"新增模板" | 手动创建或导入JSON |
| **编辑模板** | 点击模板卡片"编辑" | 修改模板信息和字段 |
| **删除模板** | 点击模板卡片"删除" | 有二次确认提示 |
| **JSON导入** | 点击"导入JSON文件"或"手动输入JSON" | 支持文件和文本输入 |

## 💡 使用技巧

1. **快速创建**：复制 `template.json` 内容，点击"手动输入JSON"粘贴
2. **字段命名**：key建议使用英文小写+连字符，如 `product-name`
3. **模板ID**：建议使用有意义的标识符，如 `product-price-tag`
4. **屏幕类型**：根据实际设备选择，影响显示效果

## 🎯 常见用例

### 用例1：创建商品价格标签
1. 模板ID: `product-price`
2. 名称: `商品价格标签`
3. 字段: 
   - 商品名称 (text, 必填)
   - 价格 (text, 必填, 默认值: `¥0.00`)
   - 条形码图片 (image, 非必填)

### 用例2：创建库存标签
1. 模板ID: `inventory-tag`
2. 名称: `库存管理标签`
3. 字段:
   - 商品编号 (text, 必填)
   - 库存数量 (text, 必填)
   - 库位代码 (text, 必填)

## ❓ 常见问题

**Q: 为什么模板列表为空？**
A: 请检查：
1. 后端服务是否正常启动
2. API接口 `/api/v1/templates` 是否可访问
3. 数据库是否有模板数据

**Q: JSON导入失败怎么办？**
A: 检查：
1. JSON格式是否正确
2. 是否包含必需的 `tid` 和 `tname` 字段
3. 如果使用简单格式，确保有 `data` 字段

**Q: 字段在模板更新中不显示？**
A: 确保：
1. 模板已成功保存
2. 字段的 `required` 属性正确设置
3. 刷新模板更新页面
